Periodic functions

A function `f` is `2m`-periodic `<=>` `f(x + 2m) = f(x)` `AA x in RR`. `f(x)` is defined `AA x in [m, -m]` and `f` is `2m` periodic `=>` `f` is defined `AA x in RR`. Sinusoids are the quintessential example of a periodic function. In fact, any expression of the form `sum_(n in ZZ) (\ a_n cos(nx) + b_n sin(nx)\ )` is `2pi`-periodic (note that `n in ZZ`; rational, non-integer coefficients of `x` would also yield periodic functions, just not of period `2pi`). This makes sense intuitively since the sum of two `2m`-periodic functions will also be `2m`-periodic.

Trigonometric series and Fourier series

A series of the form `1/2 a_0 + sum_(n=1)^(oo) (\ a_n cos(nx) + b_n sin(nx)\ )` is said to be a trigonometric series. `AA x` for which a trigonometric series converges, it defines a function `f(x) := \ \ 1/2 a_0 + sum_(n=1)^(oo) (\ a_n cos(nx) + b_n sin(nx)\ )` Such a trigonometric series is said to be the Fourier series for `f`. For any given Fourier series, given the above formulation, `f(x) = f(x + 2pik), \ \ \ k in ZZ`, `AA x in S`, where `S sube RR` is the set of values of `x` for which the trigonometric series converges.

Kronecker delta

`delta_(mn) = {(1,if m = n), (0, if m != n):}, \ \ \ \ AA m,n in ZZ` This is essentially the mathematical equivalent an equality operator (`"=="`). Using this operator, we may derive the following results: `int_(-pi)^pi sin(mx) \ sin(nx) \ dx = pi delta(mn)` `int_(-pi)^pi cos(mx) \ cos(nx) \ dx = pi delta(mn)` `int_(-pi)^pi sin(mx) \ cos(nx) \ dx = 0` (notice the parity of each integrand) N.B. be sure not to confuse this with the Dirac delta, which is often used to define the DFT.
