Vector space

  Click here for the definition of a vector space. One can check that a set `W` is a vector space by checking both 1.     that it is closed under vector addition and scalar multiplication 2.     that it satisfies the eight axioms of a vector space. One can check that `\ V: \ V sube W` is a subspace (and thus a vector space) by checking both 1.     that it is closed under vector addition and scalar multiplication 2.     `bb 0 in V`     (N.B.   `V` is non-empty `=>` `bb 0 in V`)  

Linear independence, spanning, dimension

 

Linear independence

A finite list `L = bb v_1, ..., bb v_m` is linearly independent iff `lambda_1 bb v_1 + ... + lambda_m bb v_m = bb 0`     `=>`     `lambda_1 = lambda_2 = ... = lambda_m = 0`  

Spanning

  `S` spans `V` if every element of `V` can be written as a linear combination of elements of `S`. If `V` has a finite spanning set then we say `V` is finite-dimensional, and the dimension of `V` is the number of elements in a basis for `V`. If finite spanning set `S` spans `V`, and `S` forms a linearly independent list, then `S` is a basis for `V`.
