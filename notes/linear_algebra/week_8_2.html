The product of permutations

Let `alpha = (\ 1 \ \ 2 \ \ 3 \ \ 4 \ ),`   `beta = (\ 1 \ \ 3 \ \ 2 \ \ 4 \ ).`   `alpha beta = (\ 1 \ \ 2 \ \ 3 \ \ 4 \ ) \ (\ 1 \ \ 3 \ \ 2 \ \ 4 \ )`   ` = (\ 1 \ \ 4 \ \ 2 \ ) \ (\ 3 \ ) = (\ 1 \ \ 4 \ \ 2 \ )`   N.B.   `(beta alpha)^(-1) = alpha^(-1) beta^(-1)`  

Transposition representation

Any permutation can be written as a product of (non-disjoint) transpositions (similar to how any permutation can be written as a product of disjoint cycles). [ example ] `(\ a_1 \ \ a_2 \ \ a_3 \ \ ... \ \ a_r \ ) = (\ a_1 \ \ a_2 \ ) \ (\ a_2 \ \ a_3 \ ) \ ... \ (\ a_(r-1) \ \ a_r \ )`  

Parity/sign

permutation `alpha` is even     `<=>`     `alpha` can be written as the product of an even number of transpositions   permutation `alpha` is odd     `<=>`     `alpha` can be written as the product of an odd number of transpositions N.B.   these two properties are mutually exclusive (intuitively). We may now define the sign of a permutation `"sg"(alpha) = {(\ \ \ 1, " if the permutation is " bb "even"), (\ \ \ -1, " if the permutation is " bb "odd"):}`   N.B.     `"sg"(alpha beta) = "sg"(alpha)\ "sg"(beta)`     and     `"sg"(alpha^(-1)) = "sg"(alpha)`  

The symmetric group

The set of all permutations of `T_n` is called the symmetric group on `n` objects, written `S_n`. The following properties hold for the symmetric group `alpha in S_n " and " beta in S_n`     `=>`     `alpha beta in S_n`   `(alpha beta) gamma = alpha (beta gamma)`     `AA \ \ alpha, beta, gamma in S_n`   `e alpha = alpha e = alpha`     `AA \ \ alpha,`     where `e` is the identity permutation   `(AA \ \ alpha in S_n) \ \ (EE \ \ beta in S_n) \ \ : \ \ alpha beta = beta alpha = e`     (i.e. any permutation in `S_n` has an inverse) There are `n!` elements (permutations) in `S_n`.  

The alternating group

The set of all even permutations on `T_n` is called the alternating group on `n` objects, written `A_n`. Intuitively, `A_n` has `(n!)/2` elements for `n>=2`   (`1` element if `n=1`).
