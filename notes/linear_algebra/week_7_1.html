Linear recurrence relations

Consider the linear recurrence relation defined by `x_0 = 1`   `x_1 = 1`   `x_n = x_(n-1) + 2x_(n-2)` We may equivalently write this as `[(x_(n-1)), (x_n)] = [(0, 1), (2, 1)][(x_(n-2)), (x_(n-1))]`   `= A[(x_(n-2)), (x_(n-1))]`   `= A^n[(x_0), (x_1)]`   `= A^n[(1), (1)],`         where `A = [(0, 1), (2, 1)].` This, of course lends itself to diagonalization, so that we might easily compute the `n`th power of matrix `A`.
`det(A - lambda I) = -(lambda + 1)(lambda - 2) = 0`     `=>`     `lambda_1 = -1, \ \ \ lambda_2 = 2` This yields the respective eigenvectors   `[(1), (-1)], \ [(1), (2)]`     `=>`     `P = [(1, 1), (-1, 2)].`  
`=>`     `A^n = PD^nP^(-1)`     (which may be easily expanded) A seperate but related use for diagonalisation can be found in the city vs country problem.  

Coupled differential equations

The following differential equations may be solved using a specific change of variables. `(dx)/(dt) = 5x + 3y`   `(dy)/(dt) = -6x - 4y`     `((x^'), (y^')) = [(5, 3), (-6, -4)][(x),(y)] = A[(x),(y)],`     for matrix `A = [(5, 3), (-6, -4)]`   `A` may now be diagonalized, yielding `P = [(1, 1), (-1, -2)]`     and     `D = [(2, 0), (0, -1)]` We now implement a change of variables   `[(x), (y)] = P\ [(X), (Y)]`   `=>`     `[(X), (Y)] = P^(-1)[(x), (y)]`   `=>`     `[(X^'), (Y^')] = P^(-1)[(x^'), (y^')]`   `= P^(-1)A[(x), (y)] = P^(-1)AP[(X), (Y)]`  
`=>`     `[(X^'), (Y^')] = D\ [(X), (Y)] = [(2X), (-Y)]`
  `=>`     `X^' = 2X`     and     `Y^' = -Y`   `=>`     `X = Ae^(2t)`     and     `Y = Be^(-t)`   `=>`     `x = Ae^(2t) + Be^(-t)`     and     `y = -Ae^(2t) - Be^(-t)`  

The Cayley-Hamilton Theorem

The Cayley-Hamilton theorem states that any square matrix `A` satisfies its own characteristic equation.   (i.e. the characteristic equation `lambda^2 - 2lambda + 1` is satisfied by `lambda = A => A^2 - 2A + I = O`).
