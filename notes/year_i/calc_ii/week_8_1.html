Exact First-Order Differential Equations

A first-order differential euqation `D`   `M(x, y) + N(x, y) (dy)/(dx) = 0` is said to be exact if `(dz)/(dx) = 0,`       with `z = g(x, y)` Given that `dz = (del g)/(del x) dx + (del g)/(del y) dy` we can derive the following
`D` is exact     `<=>`     `(del M)/(del y) = (del N)/(del x)`     `<=>`     `EE \ \ g(x,y) \ : { (\ g_x = M), (\ g_y = N) :}`
  (by Clairaut's Theorem, since   `(del^2 g)/(del y del x) = (del M)/(del y)`   and   `(del^2 g)/(del y del x) = (del N)/(del x)`     [proof of sufficiency]) We can use this to solve exact differential equations (method).
