Matrices and linear maps

 

Linear map (`RR^2 to RR^2`)

A linear map (linear transformation) from `RR^2` to `RR^2` is a map  `f: RR^2 to RR^2` of the form `[(x), (y)] |-> [(ax + by), (cx + dy)]` , which may be represented using the following matrix: `[(a, b), (c, d)]` . Then, we may define matrix-vector multiplication, such that `[(a, b), (c, d)][(x), (y)]` is equivalent to `f([(x), (y)])`.  

Linear map (`bbb F^n to bbb F^m`)

A linear map from `bbb F^n` to `bbb F^m` is a map of the shape `[(x_1), (x_2), (vdots), (x_n)] |-> [(y_1), (y_2), (vdots), (y_m)]`, where there is a fixed `m xx n` matrix `A = [a_(ij)]` over `bbb F` such that `y_i = sum_(j=1)^n a_(ij) \ x_j \ \ (i = 1, 2, ..., m)`.  

Component Notation

The following two notational devices may be used to refer to the `(i, j)`th component of matrix `A`. `i. \ \ \ [A]_(i,j)`   (the comma may be omitted if there is no danger of confusion) Let `A = [a_(ij)]`   `ii. \ \ a_(ij)`  

Addition and scalar multiplication

  `[A + B]_(ij) = [A]_(ij) + [B]_(ij)`   `[lambda A]_(ij) = lambda [A]_(ij)`   Matrices also satisfy the equivalents of the 9 vector properties covered in chapter 1.  

Matrix multiplication

  It is very useful to think of matrix multiplication as composition of linear maps. `[AB]_(ij) = sum_(k=1)^n [A]_(ik) \ [B]_(kj)` The process of matrix multiplication is perhaps best thought of as computing the dot product of the `i` th row in `A` with the `j` th column in `B`, then storing the result in `[AB]_(ij)`. N.B. matrix multiplication is not commutative. (`AB != BA` in general)

Properties

  `A(BC) = (AB)C`   (associativity) `A(B + C) = AB + AC`   (first distributive law) `(A + B)C = AC + BC`   (second distributive law)
