Transposition rules

The following properties hold for `M^T`, the transpose of matrix `M`.   `(A + B)^T = A^T + B^T`   `(AB)^T = B^T \ A^T`   N.B.   a matrix `A` is called symmetric if `A^T = A`.  

Permutations

Consider the set `T_n = {1, 2, 3, ..., n}`. Then a permutation `alpha` on the set `\ T_n` is a bijection from itself to iteself. `alpha: \ \ \ T_n -> T_n,`     `"where" \ \ alpha \ \ "is bijective"`   N.B.   permutations may be composed or inverted as with ordinary mappings   (the mapping `alpha beta` corresponds to performing `beta` followed by `alpha`). We may use the following notation to represent a permutation: `alpha = ((1, 2, 3, 4, 5, 6, 7), (2, 5, 3, 4, 6, 7, 1))`   However, this notation is a little cumbersome, and so in general it is better to use cycle notation (as described below).  

Cycles

 

Cycle

If you iteratively apply a permutation to a given element in the set, the element will be mapped to various other elements in the set, before finally returning to the original element. Intuitively, this process is referred to as a cycle. In the worst case, an element would be mapped through all the other elements in the set before it returns to its original value. This is referred to as a cycle of order `n`. `a = alpha^t (a)`  

Cycle length

The length of a cycle is the number of iterations it takes to complete a full loop (i.e. return to the original value).  

Cycle decomposition

A given set will have a finite number of cycles which each element falls into. Also, each element will fall into exactly one cycle (and thus cycles may be referred to as disjoint from each other). We can decompose a set into its constituent cycles.  

Cycle type

The cycle type is a list of the lengths of each of the cycles comprising `alpha`.  

Cycle representation of a permutation

The cycle `beta = ((1, 2, 3, 4, 5, 6), (1, 3, 6, 5, 4, 2))` may be written in cycle notation as `beta = (\ 2 \ \ 3 \ \ 6 \ )\ (\ 4 \ \ 5 \ )` where each bracket contains an individual cycle. This notation represents our permutation as a product of disjoint cycles. N.B.   Notice that `1` can be omitted from the representation, since it   forms a cycle of length one.  

Cycle order

A permutation `T_n` is said to be a cycle of order `r`, if it cyclically permutes `r` elements, and fixes all the rest. `(\ 2 \ \ 5 \ \ 3 \ )` is a cycle of order `3`.  

Special cases

 

Case `r=1`

    all the elements are fixed under `T_n`.  

Case `r=2`    

(transposition)     swaps two of the elements, and fixes all the rest.  

Case `r=n`

    all the elements in the set form one cycle under `T_n`.
