`E_(ij)` and `"diag"`

  `E_(ij)` is a matrix with `0` everywhere but the `(i,j)`th cell. e.g. `E_(12) = [(0, 1, 0), (0, 0, 0)]` Then, all `m xx n` matrices `E_(ij)` form the basis for the vector space of `m xx n` matrices. `"diag"(a_1, cdots, a_n) = [(a_1, cdots, 0), (vdots, ddots, vdots), (0, cdots, a_n)]` with `det(M) = prod_(i=0)^n \ a_i`     for any triangular matrix `M` `=> \ \ \ det("diag"(a_1, cdots, a_n)) = prod_(i=0)^n \ a_i`  

EROs

These are the three elementary row operations. (I) Multiply a row of `A` by a non-zero scalar. (II) Swap two rows of `A`. (III) Add a multiple of one row to another row of `A`. Any ERO on `A` can be achieved by computing `PA` for some suitable invertable matrix `P`. EROs preserve the row space.  

Echelon form

A matrix is in echelon form if 1. All zero rows are at the bottom. 2. If rows `i` and `i+1` are non-zero, the first non-zero entry in row `i+1` is to the right of the first non-zero entry in row `i`. In other words, the zeroes get moved over to the bottom left hand corner. A matrix is in row reduced echelon form if 1. It is in echelon form. 2. The first entry in a non-zero row must be `1`. 3. These `1`s which start each row dictate that the rest of their column be `0`. The rank of `A` is the number of non-zero rows in the row-reduced version of the matrix, which also corresponds to the dimension of the row space and column space of `A`.  

Practical applications

  Solving systems of simultaneous linear equations   Find a subset of the columns of `A` that is a basis for their span   Computing determinants   Computing inverses     (requires row-reduced echelon)
