Vector

We will be using the following notation for vectors:   `bb r(t) = x(t) \ bb i + y(t) \ bb j`   `= (\ x(t), \ \ y(t) \ )` The derivative of a vector is defined as follows: `(d bb r)/(dt) = lim_(Delta t -> 0) (bb r(t + Delta t) - bb r(t))/(Delta t)`   `= (dx)/(dt) bb i + (dy)/(dt) bb j`   `= (\ (dx)/(dt), \ \ (dy)/(dt) \ )` precisely as with scalars. When `t` represents time, `(d bb r)/(dt)` is a vector representing the velocity at time `t`. This also happens to be tangent to the curve at that point (the point `P: (x, y)`). `bb v(t) = (d bb r)/(dt)`   `v(t) = |bb v(t)|`   `bb T(t) = (bb v(t))/(v(t))`     [ (unit) tangent vector ]   `s(t) = int_(t_0)^(t_1) \ v(t) \ dt`     [ arc length (distance travelled) ]  

dr/ds

  `v(t) = (ds)/(dt) = |(d bb r)/(dt)|`   `=> 1 = |(d bb r)/(dt) div (ds)/(dt)|`     `=> v = |(d bb r)/(ds)| = 1.`  

Parameterisation in terms of `s`, arc length

We can replace our `t` parameter with any other variable to get a new set of parametric equations representing the same curve. It is sometimes useful to use the arc length `s` as our "new `t` parameter". To do this we can simply rearrange `s` for `t`, and substitue these values into `x(t)` and `y(t)` to create a new set of parametric euqations `x(s)` and `x(t)`. One useful byproduct of doing this is that, since `|d bb r // ds| = 1`, we have that `bb T(s) = (d bb r) / (ds)`. Notice that no normalisation is required!
