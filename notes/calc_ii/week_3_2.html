Representations of curves

Consider the curve `C`. There are three ways in which it can be represented.

Explicit  (`RR^2`)

  `Gamma_f = { \ (x, y) in RR^2 : \ \ y = f(x) \ }` For many such graphs, the curve can be represented by a single equation, such that `C = Gamma_f` (e.g., line, polynomial) Other curves must be represented by two seperate functions. Take for example a circle. `C_(text(UPPER)) = Gamma_(f_(text(UPPER)))`   `C_(text(LOWER)) = Gamma_(f_(text(LOWER)))` Some curves do not lend themselves to explicit representation - their equations would be incredibly complex (e.g., spiral, squiggle)

Implicit  (`RR^2`)

  `Sigma_g = { \ \ (x, \ y) in RR^2 : \ \ g \ (x, \ y) = 0 \ \ }` with solution set `C = Sigma_g`  

Parametric  (`RR^n`)

  `C = { \ \ \ \ (x, y) in RR^2 : \ \ { (\ x = f(t)), (\ y = g(t)) :} \ \ , \ \ text(with) \ \ t in (a, b), \ \ \ a, b in RR \ \ \ \ }`  

Circle

  `x = R \ \ cos(t)` `y = R \ \ sin(t)`   N.B.     `x^2 + y^2 = R` Minimum range:     `0 <= t <= pi`  

Line segment (version 1)

  `x = R \ \ cos^2(t)` `y = R \ \ sin^2(t)`   N.B.     `x + y = R` Minimum range:     `0 <= t <= pi/2`  

Line segment (version 2)

  `x = R \ \ (1-t)` `y = R \ \ t`   N.B.     `x + y = R` Range:     `0 <= t <= 1`   N.B. for the first representation of a line segment, different values of `t` oscillate up and down the curve. With the second representation, the domain of `t` determines the range of the line segment.

Hyperbola

  `x = sinh(t)` `y = cosh(t)`   N.B.     `y^2 - x^2 = 1` Range:     `-oo <= t <= +oo`  

Converting from explicit to parametric

Consider the curve `y = f(x)`. Simply `x = t` `y = f(t)`  

Polar representation

To calculate the polar representation of a spiral, note that `x = r cos theta` `y = r sin theta` By defining `r` to be a function of `theta`, we may write `r(theta) = a theta + b`   N.B. Setting `a = 0` and `b = R` gives us a circle. OR `r(theta) = a \ * \ exp \ (\ b theta \ )`
