Representations of curves

Consider the curve `C`. There are three ways in which it can be represented.

Explicit  (`RR^2`)

  `Gamma_f = { \ (x, y) in RR^2 : \ \ y = f(x) \ }` For many such graphs, the curve can be represented by a single equation, such that `C = Gamma_f` (e.g., line, polynomial) Other curves must be represented by two seperate functions. Take for example a circle. `C_(text(TOP)) = Gamma_(f_(text(top)))`   `C_(text(BOTTOM)) = Gamma_(f_(text(bottom)))` Some curves do not lend themselves to explicit representation - their equations would be incredibly complex (e.g., spiral, squiggle)

Implicit  (`RR^2`)

  `Sigma_g = { \ \ (x, \ y) in RR^2 : \ \ g \ (x, \ y) = 0 \ \ }` with solution set `C = Sigma_g`  

Parametric  (`RR^n`)

  `C = { \ \ \ \ (x, y) in RR^2 : \ \ { (\ x = f(t)), (\ y = g(t)) :} \ \ , \ \ text(with) \ \ t in (a, b), \ \ \ a, b in RR \ \ \ \ }`
