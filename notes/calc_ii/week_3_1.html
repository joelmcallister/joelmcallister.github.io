Solving complicated D.E.s using Taylor series

Consider the differential equation `x (d^2y)/(dx^2) + (dy)/(dx) + (xy) = 0`   `y(0) = 1`   `(dy)/(dx) (0) = 0` the solution to which cannot be expressed analytically. Choose `y` to be a general power series. `y = sum_(k=0)^(oo) \ a_k * x^k = a_0 + a_1 * x + a_2 * x^2 + a_3 * x^3 + ... + a_k x^k`   `= 1 + sum_(k=2)^(oo) \ a_k * x^k`   `y' = sum_(k=2)^(oo) \ k \ a_k * x^(k-1)`   `y^″ = sum_(k=2)^(oo) \ k(k-1) \ a_k * x^(k-2)`   `x y^″ + y' + xy = sum_(k=2)^(oo) \ (a_(k-2) + k^2 \ a_k) \ x^(k-1)` You now have an infinite set of equations, with which to find the coefficients `a_k`   `a_(k-2) + k^2 \ a_k`       `AA \ \ k in NN_(>=2)`   `color(lightgrey)( \ \ => \ \ \ y = J_0(x) = sum_(l=0)^(oo) (-1)^l x^(2l) / (2^l * l!)^2 \ \ )`  

L'Hopital's rule

  `lim_(x->x_0) f(x)/g(x) = lim_(x->x_0) (f'(x))/(g'(x))` This can be derived by replacing `f(x)` and `g(x)` with their Taylor series expansions and making the appropriate cancellations.
