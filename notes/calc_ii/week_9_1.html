Stationary points of `f(x, y)`

A local minimum (maximum) is a point `(x_0, y_0)` such that for all points `(x, y)` in an open disk around `(x_0, y_0)`   `f(x, y) >= \ (<=) \ f(x_0, x_y).` To be a local minimum (maximum), `(x_0, y_0)` must be a stationary point of `f(x,y)` `(x_0, y_0)` is a stationary point     `<=>`     `AA \ bb hat n \ : \ \ \ [ (del f)/(del n) ]_[\ (x_0, y_0)] = 0`     `<=>`     `grad f(x_0, y_0) = bb 0`  

Local minima (maxima) of `f(x, y)`

Using our quadratic Taylor polynomal `P_2(x, y)`, it may be derived theoretical `(x_0, y_0)` is a local minimum (maximum) of `f(x, y)`     `lArr`     `AA \ m != 0 \ : \ \ \ bb m^T \ H_f \ (x_0,y_0) \ bb m \ > \ (<)\ 0,`     with `bb m = [(x - x_0), (y - y_0)]`   N.B.   this is known as quadratic form  

Diagonal case

Consider the case when the Hessian is diagonal `H_f \ (x_0, y_0) = [(a, 0), (0, b)]`     `=>`     `bb m^T \ H_f \ (x_0,y_0) \ bb m = a \ bb m_x^2 + b \ bb m_y^2` Thus, we get a local minimum (maximum) when `a = (del^2 f)/(del x^2) > \ (<) \ 0`     and     `b = (del^2 f)/(del y^2) > \ (<) \ 0\ .` In addition `a < 0`   and   `b > 0`   or   `a > 0`   and   `b < 0`         `=>`         `(x_0, y_0)` is a saddle point   `a = 0`   or   `b = 0`     `=>`     test inconclusive  

General case

The following criteria may be derived in the general case `det H_f < 0`     `=>`     saddle point `det H_f = 0`     `=>`     inconclusive `det H_f > 0` and `b > (<) \ 0`     `=>`     local minimum (maximum)
