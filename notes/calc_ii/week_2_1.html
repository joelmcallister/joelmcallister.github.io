Taylor series

For some infinitely differentiable function `f: \ (a, b) \ sub \ text(dom.) \ \ f` we may use Taylor's method to find the power series `S_f\ (x)` which converges to `f`. Recall the general definition of a power series `S_f \ (x) = sum_(k=0)^(oo) a_k \ (x - x_0)^k` To find a power series which converges to `f`, we require that `S_f^((l)) (x_0) = f^((l)) (x_0), \ \ \ AA l in ZZ_(>= 0)` for fixed point   `x_0`. We may also derive that `S_f^((l)) (x_0) = l! \ a_l, \ \ \ AA l in ZZ_(>=0)` Therefore `a_l = f^((l)) (x_0)/(l!)` With this, we can write our Taylor series `S_f\ (x)` generated by `f`  
`S_f \ (x) = sum_(k=0)^(oo) \ a_k \ (x - x_0)^k`   `= sum_(k=0)^(oo) \ (f^((k)) (x_0) )/(k!) \ (x - x_0)^k`   `= f(x_0) + sum_(k=1)^(oo) \ (f^((k)) (x_0) )/(k!) \ (x - x_0)^k`
  N.B. it is good practice to write our Taylor series in the last form, since our `a_k` coefficient formula may not generalise to `f(x_0)` (e.g., we may end up with something like `-1!` in our expression) For a very broad class of functions (and all the function we will be interested in in this module) we have that For all analytic functions, we have that `S_f \ (x) = f(x) \ \ \ \ \ AA x in \ text(RoC) \ \ S_f` . In this module, we are only interested in analytic functions. N.B. the Taylor series of an odd/even function will only contain odd/even contributions (respectively). This also applies to Taylor polynomials.
