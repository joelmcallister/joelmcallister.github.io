Preliminary remark

  `|x| < 1 \ => \ lim_(n->oo) x^n = 0`  

Series

A series `S` is an infinite sum, defined as `S = sum_(k=0)^(oo) b_k`  

Convergence

A partial sum of a series is the sum of the first `n` terms of that series `S_n = sum_(k=0)^(n-1) b_k`   `S` converges to finite value `v`   `<=>`   `lim_(n->oo) S_n = v`   `S` does not converge to a finite value   `<=>`   `lim_(n->oo) S_n` does not exist  

Example: geometric series

The following trick can be used to find the value to which a series converges.   `S(x) = sum_(k=0)^(oo) x^k` `=> S_n(x) = sum_(k=0)^(n-1) x^k`   `=> S_n(x) * (1-x)`   `= (1 + x + x^2 + x^3 + ... + x^(n-1)) (1 - x)`   `= 1 - x^n`   `=> S_n(x) = (1 - x^n)/(1 - x)`   (`x != 1`)  

Case `|x| < 1`

  `lim_(n->oo) S_n(x) = (1 - lim_(n->oo) x^n) / (1 - x) = 1/(1-x)`   `=> \ S(x)` converges to `1/(1-x)`.  

Case `|x| >= 1`

  `S` diverges.
 

Example: integrating geometric series

  `int_0^y S_n(x) \ dx \ = \ int_0^y (sum_(k=0)^(n-1) x^k) \ dx` We may now attempt term-by-term integration. N.B. that this will only work if all of the resulting integrals converge. `= sum_(k=0)^(n-1) int_0^y x^k \ dx \ = \ sum_(k=0)^(n-1) y^(k+1)/(k+1)`. Let `l = k+1`. `int_0^y S_n(x) \ dx \ = \ sum_(l=1)^(n) y^l / l`. We may consider this as the partial sum `T_n(y)` of the sum `T(y)`   `=> T_n(y) = sum_(l=1)^(n) y^l / l \ = \ int_0^y (1-x^n)/(1-x) \ dx = -ln(1-y) - int_0^y x^n/(1-x) \ dx` It may be shown that if   `|y| < 1`,     `int_0^y x^n//(1-x) \ dx -> 0`   as   `x -> oo`. Therefore, `|y| < 1 \ => \ T(y)`   converges to   `-ln(1-y)`.  

A series for natural log

Using the above result, and setting `x = -y`, we may write `ln(1+x) = - sum_(l=1)^(oo) (-x)^l/l \ = x - x^2/2 + x^3/3 - x^4/4 + ...`   N.B. if you plug in the value `x = -1`, you get the harmonic series (or rather, the harmonic series multiplied by `-1`). This follows, since the harmonic series is divergent, and `log(0)` is undefined.

The harmonic series
