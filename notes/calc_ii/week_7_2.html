Parametric equation as a contour

Consider the curve     `C = { \ (x, y): \ \ g(x, y) = c \ }`   Then there is a vector `bb r` such that `bb r = (x, y) = (\ x(t), y(t)\ )`   `=> AA \ t: \ \ \ z(t) = g(x(t), y(t)) = c` It may be derived that `grad g * bb T = 0` In other words,   `bb T`   is orthogonal to   `grad g`   everywhere on `C`. Therefore, `grad g` is normal to the curve at `P(x, y)`. We can thus use `grad g` to determine our unit normal vector `bb N`. //     TODO     add the definitions below for `bb N` and `K` to week 4 notes

Obtaining `bb N` from `grad g`

  `bb N = +-(grad g)/|grad g| = +- 1/|grad g| \ [(g_x), (g_y)]`   N.B.   the sign must be chosen so that `bb N` points towards the center of curvature.  

The Hessian - obtaining `K` from `grad g`

The Hessian is a `2 xx 2` matrix whose elements are the second order partial derivatives of `g(x, y)`. `H_g = [(g_(x x), g_(xy)), (g_(yx), g_(yy))]`   N.B.     `g_(xy) = g_(yx)` Switching to column vectors, we may write `bb T = [(T_x), (T_y)] = +- 1/|grad g| \ [(-g_y), (g_x)]`   `=> bb T^T = [T_x, T_y]` Then, we may define `K` as follows `K = 1/|grad g| \ |\ bb T^T \ H_g \ bb T \ |`
