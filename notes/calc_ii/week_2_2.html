Taylor polynomials

The coefficients of a Taylor series do not always arise in a patterned way. In this case, the best we can do is compute the first `n` terms of the series, producing what is referred to as a Taylor polynomial. `P_n(x) = f(x_0) + sum_(k=1)^n (f^((k))(x_0))/(k!) (x - x_0)^k`   Case: `n=1` The "linearisation" of `f` about `x_0`. `P_1(x) = f(x_0) + f'(x_0) * (x - x_0)` This equation represents the tangent line to the function at `x_0`. This is because `P_1(x_0) = f(x_0)` and `P_1'(x_0) = f'(x_0)`, by the definition of a Taylor series.   Case: `n=2` The parabola which best fits the function at `x_0`. `P_1(x) = f(x_0) + f'(x_0) * (x - x_0) + 1/2 f^″(x_0) * (x - x_0)^2`  

Taylor's theorem

  `f(x) = sum_(k=0)^(oo) \ (f^((k)) (x_0) )/(k!) \ (x - x_0)^k`   `= sum_(k=0)^n \ \ ... \ \ + \ sum_(k=n+1)^(oo) \ (f^((k)) (x_0) )/(k!) \ (x - x_0)^k`   `= P_n(x) + R_(n+1)(x)` where `R_(n+1)(x) = sum_(k=n+1)^(oo) \ (f^((k)) (x_0) )/(k!) \ (x - x_0)^k`  

Taylor's theorem

  `EE \ epsilon in RR_(>0), \ M in RR_(>0) \ :`   `|x - x_0| < epsilon \ \ \ => \ \ \ |R_(n+1)(x)| <= M * (x-x_0)^(n+1)` In other words, as `x` tends to `x_0`, `R_(n+1)` tends to `0` at least as fast as `(x - x_0)^n`. This can be expressed in big-O notation, as such: `R_(n+1)(x) = O(\ (x-x_0)^(n+1)\ )` One may then write, for example `e^x = 1 + x + O(x^2)` meaning that the error involved in the approximation of `e^x` with `1 + x` is of the order `x^2`. As is customary with big-O notation, superfluous constants and coefficients may be removed.
