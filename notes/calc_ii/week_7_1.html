Directional derivative

Let `bb hat n` be a unit vector (the `hat("hat")` indicates it is a unit vector). `bb hat n = (n_x, n_y),`       `sqrt(n_x^2 + n_y^2) = 1` Then the directional derivative of `f` is `(del f)/(del n)(x_0, y_0) = d/dt[\ f(x_0 + t n_x, \ y_0 + t n_y)\ ]_(t=0)` Given that `dz = (del z)/(del x) dx + (del z)/(del y) dy`     and     `dx = dt \ n_x, \ \ \ dy = dt \ n_y` we have that `dz = ((del f)/(del x) n_x + (del f)/(del y) n_y) \ dt`  
`=> (del f)/(del n) = (del f)/(del x) n_x + (del f)/(del y) n_y`
This is our formula to calculate any given directional derivative, in the direction of a given unit vector. Note, `bb hat n` must be a unit vector - that is, `bb hat n` must be normalised (`|bb hat n| = 1`).  

Gradient

The gradient of `f` is a vector, defined as follows `"grad" \ \ f = grad f = ((del f)/(del x), (del f)/(del y)) = (f_x, f_y)` The following property is true of the gradient
`(del f)/(del n) = grad f \ * \ bb hat n`
 

Direction of steepest ascent

The direction of steepest ascent of `f(x, y)` at `(x_0, y_0)` is the direction of `bb hat n` for which `(del f)/(del n)` is maximal. `bb hat n_f \ : \ AA \ bb hat n \ \ \ (del f)/(del n) <= (del f)/(del n_f)`  
`\ bb hat n_f = (grad f)/|grad f|`       and       `(del f)/(del n_f) = |grad f|`
  (`=> bb hat n_f = grad f // (del f)/(del n_f)` ) Remarkably, this theorem says that the direction of steepest ascent can be obtained by simply normalising the gradient. In addition, the directional derivative of the direction of steepest ascent is simply the modulus of this gradient (proof).
